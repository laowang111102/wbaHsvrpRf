# 项目简介 python981

本项目是一款基于Python的程序可视化打包工具，旨在帮助用户便捷地将Python项目打包为独立可执行的exe文件，并优化程序结构及性能。

## 技术栈

- Python

## 功能模块

### 单文件封装

将Python项目打包成独立的exe程序，用户无需安装Python环境即可运行。

### 智能依赖优化

- 自动识别并分离大型库
- 打包为外部DLL文件
- 减少主程序体积，提升启动速度

### 资源文件集成

支持嵌入各类资源文件，如图片、数据文件等，确保程序运行时能正确访问所需文件。

### 自定义程序图标

允许设置专属.ico图标，使得生成的EXE文件更具辨识度和专业性。
